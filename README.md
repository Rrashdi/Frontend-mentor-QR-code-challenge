### What I learned


- In CSS, if you set the body height to 100vh, any borders or margin will increase the viewport height.
 for example, 
```css
.body {
  height: 100vh;
  margin: 2.5rem;
  border: 5px solid black;
}
```
  the total size of view port = 100vh + 2.5rem + 5px.
